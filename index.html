Terms and Concepts

  • JavaScript

    • An object-oriented computer programming language commonly used to create interactive effects within web browsers.

  • Operators

    • Are used to assign values, compare values, perform arithmetic operations, and more.

  • Variables

    • Are containers for storing data values.

  • Variable naming conventions

    • Style guidelines for programming. They typically cover: naming and declaration rules for variables and functions.

  • Functions

    • A function is a subprogram designed to perform a peticular task and are executed when called.

  • Methods

    • Actions that can be performed on objects and properties containing a function definition.

  • Parameters

    • Variables listed as part of the function definition.

  • Return

    • The values returned from a function.

  • Chaining Methods

    • Also known as cascading, it refers to repeatedly calling one method after another on an object in one continuous line of code.

  • Strings

    • A textual data contained within quotes.

  • Booleans

    • Represents two values: True and False.

  • Undefined

    • If no return value is specified, it returns as undefined.

  • NaN

    • Represents Not a Number.

  • Escape

    • the escape character allows you to output characters normally unavailable; ie: backslash (\)).

  • Alert

    • Displays an alert box that displays a message and an "O.K." Button.

  • Comments

    • Used to make Javascript more readable and explains code to other users.

  • JQuery

    • Concise and fast Javascript Library that can be used to simplify event handling, HTML document traversing, Ajax interactions and animation for speedy website development.

  • Attributes

    • Can be used to reference elements.

  • Arguments

    • Arguments are values passed to the function when it is invoked.
